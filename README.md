# Diagonal Tile Indicators

A RuneLite plugin that displays tile indicators on diagonal tiles from the player's true tile position.

## Features

- Highlights tiles diagonally adjacent to your character's true tile
- Configurable tile range (1-10 tiles out in each diagonal direction)
- Customizable fill and border colors with alpha transparency
- Adjustable border width
- Toggle fill and border visibility independently

## Configuration

| Option | Description | Default |
|--------|-------------|---------|
| Tile Range | How far out diagonal tiles are displayed | 1 |
| Fill Color | The fill color of tile indicators | Cyan (50% opacity) |
| Border Color | The border color of tile indicators | Cyan |
| Border Width | Thickness of the tile border | 2 |
| Show Fill | Toggle fill visibility | On |
| Show Border | Toggle border visibility | On |

## Use Cases

- PvP: Visualize diagonal movement options
- PvM: Better understand tile-based mechanics
- General: Improve spatial awareness of your character's position

## Installation

Install via the RuneLite Plugin Hub by searching for "Diagonal Tile Indicators".
